# Film/Dizi İzleme Merkezi (Demo)

Bu proje **telifli film/dizi yayını yapmaz**.
- Trailer (YouTube) oynatır
- Resmî platform/IMDb/JustWatch linklerine yönlendirir
- Favoriler ve giriş bilgileri **localStorage**'a kaydedilir

## Çalıştırma (en kolay)
1) VS Code ile klasörü aç
2) `index.html` üzerinde sağ tık → **Open with Live Server**
3) Giriş yap → uygulama açılır

## Not
- Listeyi `assets/js/data.js` içinden düzenleyebilirsin.
- İstersen sonraki adım: **TMDB API** ile gerçek “trend/top” verilerini çekebiliriz (API key gerekir).
